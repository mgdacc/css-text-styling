La Familia Tipográfica

Define qué fuente se usará. Siempre se recomienda incluir una "pila" (stack) de fuentes por si la primera no carga.

body {
  /* Intenta usar Helvetica, si no está, usa Arial, 
     si no, usa cualquier sans-serif del sistema */
  font-family: "Helvetica Neue", Arial, sans-serif;
}

h1 {
  font-family: "Georgia", serif;
}
2 / 16

Tipos Genéricos de Fuentes

Son los mecanismos de seguridad de CSS. Deben ir siempre al final de tu declaración font-family.

font-family: serif;      /* Tienen remates (ej: Times New Roman) */
font-family: sans-serif; /* Sin remates, modernas (ej: Arial) */
font-family: monospace;  /* Ancho fijo (ej: Consolas, Código) */
font-family: cursive;    /* Estilo manuscrito (ej: Comic Sans) */
font-family: fantasy;    /* Decorativas (Poco uso profesional) */
3 / 16

Grosor de la Fuente (Weight)

Controla qué tan "gruesa" o "fina" se ve la letra. Puede usar palabras clave o números (100 a 900).

p {
  font-weight: normal; /* Equivale a 400 */
}

strong {
  font-weight: bold;   /* Equivale a 700 */
}

h1 {
  font-weight: 900;    /* Extra Bold / Black */
}
4 / 16

Estilo de Fuente (Style)

Principalmente utilizado para poner el texto en cursiva.

em {
  font-style: italic;
}

/* 'oblique' es similar a italic, pero es una 
   inclinación artificial si la fuente no tiene cursiva real */
.inclinado {
  font-style: oblique; 
}
5 / 16

Alineación de Texto

Controla la alineación horizontal del texto dentro de su contenedor.

h1 {
  text-align: center;
}

p {
  text-align: justify; /* Crea bordes rectos a izq y der */
}

.nota {
  text-align: right;
}
6 / 16

Decoración de Texto

Se usa para líneas decorativas. Muy común para eliminar el subrayado de los enlaces.

a {
  text-decoration: none; /* Quita el subrayado */
}

del {
  text-decoration: line-through; /* Tachado */
}

u {
  text-decoration: underline; /* Subrayado */
}
7 / 16

Transformación de Texto

Cambia las mayúsculas y minúsculas visualmente (sin cambiar el HTML).

h2 {
  text-transform: uppercase; /* TODO MAYÚSCULAS */
}

.usuario {
  text-transform: capitalize; /* La Primera Letra Mayúscula */
}

.pequeno {
  text-transform: lowercase; /* todo minúsculas */
}
8 / 16

Altura de Línea (Line Height)

Fundamental para la legibilidad. Define el espacio vertical entre líneas de texto. Se recomienda usar valores sin unidad.

p {
  /* 1.5 veces el tamaño de la fuente actual. 
     Es el estándar de oro para párrafos legibles. */
  line-height: 1.5; 
}

h1 {
  /* Títulos grandes necesitan menos altura de línea */
  line-height: 1.1;
}
9 / 16

Espaciado de Letras y Palabras

Ajuste fino del espacio horizontal (Tracking).

h3 {
  /* Espacio entre caracteres */
  letter-spacing: 2px;
}

p {
  /* Espacio entre palabras */
  word-spacing: 5px;
}
10 / 16

Sombra de Texto (Text Shadow)

Añade sombras a las letras. Sintaxis: offset-x offset-y blur color.

h1 {
  /* 2px derecha, 2px abajo, 4px difuminado, negro */
  text-shadow: 2px 2px 4px rgba(0,0,0,0.5);
}

.neon {
  /* Efecto brillo */
  text-shadow: 0 0 10px #4fc3f7;
}
11 / 16

Texto Desbordado (Truncar)

Un truco esencial para cortar texto largo con "tres puntos" (...).

.tarjeta-titulo {
  white-space: nowrap;      /* No saltar de línea */
  overflow: hidden;         /* Ocultar lo que sobra */
  text-overflow: ellipsis;  /* Poner los "..." */
  
  width: 200px;
}

Web Fonts (Google Fonts)

No estás limitado a las fuentes del sistema. Puedes importar fuentes externas.

/* En tu CSS (al principio) */
@import url('https://fonts.googleapis.com/css2?family=Roboto');

/* O en tu HTML (recomendado por performance) */
<link href="https://fonts..." rel="stylesheet">

/* Uso */
body {
  font-family: 'Roboto', sans-serif;
}
13 / 16

Atajo: La propiedad `font`

Puedes resumir muchas propiedades en una sola línea. El orden importa: style weight size/line-height family.

/* Manera larga */
p {
  font-style: italic;
  font-weight: bold;
  font-size: 16px;
  line-height: 1.5;
  font-family: Arial, sans-serif;
}

/* Manera corta (Shorthand) */
p {
  font: italic bold 16px/1.5 Arial, sans-serif;
}
14 / 16

Unidades Relativas en Texto

Es mejor usar unidades relativas para accesibilidad.

html {
  font-size: 16px; /* Base */
}

h1 {
  font-size: 2rem;   /* 2 x 16px = 32px */
}

p {
  font-size: 1rem;   /* 16px */
}
/* Si el usuario cambia el tamaño base en su navegador,
   todo el sitio se escala proporcionalmente. */

Conclusión

El texto es el 90% de la web. Una buena tipografía mejora la experiencia de usuario (UX) y la accesibilidad.

// Resumen:
1. Elige una buena familia tipográfica.
2. Asegura un buen contraste de color.
3. Usa 'line-height' generoso para párrafos.
4. No abuses de las mayúsculas ni del centrado.
